# Star Wars Opening Crawl

Exercice front-end consistant à reproduire l’animation d’ouverture de Star Wars.

## Objectif
Mettre en œuvre une animation CSS inspirée d’un rendu existant, dans une logique d’apprentissage et de révision.

## Fonctionnalités
- Animation du texte en perspective
- Défilement automatique
- Mise en scène immersive

## Technologies
- HTML
- CSS (animations)
- JavaScript

## Contexte
Projet personnel réalisé dans un cadre de révision front-end.
